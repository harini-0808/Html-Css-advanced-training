Types of tags in html

Container tags which don't have their own content : table, body, head, div
Standalone tags which can't be closed : br, img, hr
Content tags which must be opened and closed the content is entered betweeen opening and closing : h1, h2 etc
Parent and child tags : table, tr, td
Tags have attributes : id, class, href etc
